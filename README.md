## Project Overview

This project is a Personal Portfolio Website developed as part of the Thiranex Internship Task 1. The website showcases my profile, skills, projects, and contact information through a responsive and user-friendly interface. A backend API is also implemented using Node.js and Express.js to demonstrate full-stack development concepts.

---

## Features

* Personal Profile Section
* About Me Section
* Skills Showcase
* Projects Section
* Contact Form
* Responsive Design
* Backend API using Express.js
* Project Data Endpoint

## Installation and Setup

### Install Dependencies

```bash
cd backend
npm install
```

### Run the Server

```bash
node server.js
```

Server runs at:

```text
http://localhost:5000
```

## API Endpoint

### Get Projects

```text
GET /projects
```

Example:

